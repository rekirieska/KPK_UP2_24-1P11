### Вариант №22. Сервис временных слотов (Timeslot)

#### Функционал сервиса

Сервис хранит расписание занятий и перемен (начало/конец) для **учебных групп и подгрупп** в разные дни недели.  
Подгруппы могут иметь собственное расписание (например, для разных языков или физкультуры).

**Зависимости от других сервисов (заглушки):**
- `Group Service` (уровень 2) – идентификаторы групп.
- `Subgroup Service` (уровень 2) – идентификаторы подгрупп. Значение `0` означает основную группу.

**Внутренние сущности:**
- `DayOfWeek` – справочник дней недели (1 = Пн … 7 = Вс).
- `Schedule` – расписание для пары (группа, подгруппа) в конкретный день недели.
- `Timeslot` – временные слоты (занятие или перемена) внутри расписания.

#### Список функций

**DayOfWeek (справочник дней недели)**
- `create_day_of_week(day_number)` – создание дня недели.
- `get_day_of_week(id)` – получение дня недели по ID.
- `list_day_of_week()` – получение списка всех дней недели.

**Schedule (расписание звонков для группы/подгруппы)**
- `create_schedule(external_group_id, external_subgroup_id, day_of_week_id)` – создание расписания.
- `update_schedule(id, external_group_id, external_subgroup_id, day_of_week_id)` – изменение расписания.
- `delete_schedule(id)` – удаление расписания.
- `get_schedule(id)` – получение расписания по ID.
- `list_schedules(external_group_id, external_subgroup_id, day_of_week_id)` – получение списка расписаний с фильтрацией.

**Timeslot (временные слоты – занятия и перемены)**
- `create_timeslot(schedule_id, order_number, is_lesson, start_time, end_time)` – создание слота.
- `update_timeslot(id, schedule_id, order_number, is_lesson, start_time, end_time)` – изменение слота.
- `delete_timeslot(id)` – удаление слота.
- `get_timeslot(id)` – получение слота по ID.
- `list_timeslots(schedule_id, external_group_id, external_subgroup_id, day_of_week_id, order_number, is_lesson)` – получение списка слотов с фильтрацией.

#### Правила валидации данных

- Для временного слота: `start_time` < `end_time`; `order_number` > 0; комбинация (`schedule_id`, `order_number`) уникальна.
- Для расписания: комбинация (`external_group_id`, `external_subgroup_id`, `day_of_week_id`) уникальна.
- `external_group_id` проверяется заглушкой (допустимые значения 1–20).
- `external_subgroup_id`: `0` означает основную группу, от 1 до 50 – реальные подгруппы.
- `day_of_week_id` должен существовать в таблице `DayOfWeek` (1..7).

#### Создание расписания (Schedule)

Информация требуемая для создания расписания

| Параметр                | Обязательность | Тип    | Ограничение                                | Значение по умолчанию |
|-------------------------|----------------|--------|--------------------------------------------|-----------------------|
| external_group_id       | Обязательно    | Целое  | 1..20 (существующая группа)                | —                     |
| external_subgroup_id    | Обязательно    | Целое  | 0 – группа целиком, 1..50 – подгруппа      | —                     |
| day_of_week_id          | Обязательно    | Целое  | 1..7 (день недели)                         | —                     |

Уникальные комбинации: (external_group_id, external_subgroup_id, day_of_week_id)

Выходные данные

| Параметр                | Тип    |
|-------------------------|--------|
| id                      | Целое  |
| external_group_id       | Целое  |
| external_subgroup_id    | Целое  |
| day_of_week_id          | Целое  |

#### Создание временного слота (Timeslot)

Информация требуемая для создания временного слота

| Параметр      | Обязательность | Тип         | Ограничение                                    | Значение по умолчанию |
|---------------|----------------|-------------|------------------------------------------------|-----------------------|
| schedule_id   | Обязательно    | Целое       | Существующий id расписания                     | —                     |
| order_number  | Обязательно    | Целое       | Положительное, уникально для данного schedule  | —                     |
| is_lesson     | Обязательно    | Логический  | True – занятие, False – перемена               | —                     |
| start_time    | Обязательно    | Время       | Формат HH:MM:SS, меньше end_time               | —                     |
| end_time      | Обязательно    | Время       | Формат HH:MM:SS, больше start_time             | —                     |

Уникальные комбинации: (schedule_id, order_number)

Выходные данные

| Параметр      | Тип         |
|---------------|-------------|
| id            | Целое       |
| schedule_id   | Целое       |
| order_number  | Целое       |
| is_lesson     | Логический  |
| start_time    | Время       |
| end_time      | Время       |

#### Изменение временного слота по ID

Информация требуемая для изменения временного слота по ID

| Параметр      | Обязательность | Тип         | Ограничение                                    | Значение по умолчанию |
|---------------|----------------|-------------|------------------------------------------------|-----------------------|
| schedule_id   | Опционально    | Целое       | Существующий id расписания                     | —                     |
| order_number  | Опционально    | Целое       | Положительное, уникальность в рамках расписания| —                     |
| is_lesson     | Опционально    | Логический  | True – занятие, False – перемена               | —                     |
| start_time    | Опционально    | Время       | Меньше end_time                                | —                     |
| end_time      | Опционально    | Время       | Больше start_time                              | —                     |

Выходные данные – те же, что при создании.

#### Удаление временного слота по ID

Вернет `True`, если слот был удалён, иначе `False`

#### Получение временного слота по ID

Выходные данные – те же поля, что и при создании.

#### Получение списка временных слотов по заданным параметрам

Информация требуемая для получения списка

| Параметр                | Тип         | Описание                                            |
|-------------------------|-------------|-----------------------------------------------------|
| schedule_id             | Целое       | Фильтр по id расписания (опционально)               |
| external_group_id       | Целое       | Фильтр по группе (опционально)                      |
| external_subgroup_id    | Целое       | Фильтр по подгруппе (0 – группа, 1..50)            |
| day_of_week_id          | Целое       | Фильтр по дню недели (опционально)                  |
| order_number            | Целое       | Фильтр по порядковому номеру (опционально)          |
| is_lesson               | Логический  | Фильтр по типу: занятие или перемена (опционально)  |

Выходные данные – список временных слотов (те же поля, что и при создании).
